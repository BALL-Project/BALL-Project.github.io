Detailed Description

Line class. An instance of Line represents an instance of the geometric representation "line". A line has the following properties.

  • color - the color of the line
  • vertex1 - the first vertex of the line
  • vertex2 - the second vertex of the line
    The class Line is derived from the classes GeometricObject and Vertex2. See these classes for further information concerning interface and additional methods.
